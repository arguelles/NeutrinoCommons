Module xsections

source code


Author  : C.A. Arguelles
Date    : 10/MAY/2011

This script contains information of the neutrino
cross sections, including differential cross sections.
It also implements the NUSIGMA interface.

Log:

- Modified on 9/FEB/2012 : adding routines by M. Bustamente
to include neutrion-nucleus interactions.

dsde_CC_NusigmaBin(E_1, E_2, param)

source code 

Uses the FORTRAN bin to calculate the neutrino/antineutrino differential charge current cross section. The FORTRAN executable was generated using the NUSIGMA FORTRAN library.

Implements:

ds/dE in nu + N -> l + X,

assuming isoscalar target.

Note: This function calls DIRECTLY the FORTRAN bin. Function replaced by the faster NUSIGMA-interface.

Parameters:
  • E_1 (float) - neutrino energy [GeV]
  • E_2 (float) - lepton energy [GeV]
  • param (physics parameter set) - physics parameter set
Returns: float
neutrino differential CC cross section. [cm^2 GeV^-1]

dsde_CC_NusigmaBin_ListOpt(E_1, E_2, param, flv=0)

source code 

Uses the FORTRAN bin to calculate the neutrino/antineutrino differential charge current cross section. The FORTRAN executable was generated using the NUSIGMA FORTRAN library. Optimized for several calls of the same E_1.

ds/dE in nu + N -> l + X

Assuming isoscalar target. The neutrino flavor is specified by flv.

Parameters:
  • E_1 (float) - neutrino energy [GeV]
  • E_2 (float) - lepton energy [GeV]
  • param (physics parameter set) - physics parameter set
  • flv (integer) - 0 : e, 1 : mu, 2 : tau
Returns: float

dsde_CC_opt(E_1, E_2, param, flv=0, neutype=None, optimize='neutrino')

source code 

Calculates differential neutrino CC DIS cross section. Optimized for several calls of the same E_1.

Updated : E_2 optimization mode added.

ds/dE in nu + N -> l + X

Assuming isoscalar target.

Parameters:
  • Enu_1 (float) - neutrino energy [GeV]
  • Enu_2 (float) - lepton energy [GeV]
  • param (physics parameter set) - physics parameter set
  • flv (integer) - 0 : e, 1 : mu, 2 : tau
  • neutype (integer) - 0 : neutrino, 1 : antineutrino; Overwrites param.neutype.
  • neutype (integer) - neutrino : optimize for E_1, lepton : optimize for E_2
Returns: float
neutrino differential CC DIS cross section. [cm^2 GeV^-1]
