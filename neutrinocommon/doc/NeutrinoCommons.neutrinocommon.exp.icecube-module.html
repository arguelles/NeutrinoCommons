NevtSpectra(flux, neutype, T, param)

source code 

Calculates event number spectra for UP going neutrinos.

Spectra:

dPhi/dE_nu = T * Phi_nu * cross_section * nucleon_num

where

nucleon_num = (Aeff_mu * R_muon)*density*Na

Parameters:
  • flux (flux object) - flux.neuflux(E,neutype) returns flux in [GeV^-1 m^-2 s^-1] or [m^-2 s^-1]
  • flavor (integer) - neutrino flavor (0:e,1:mu,2:tau)
  • T (float) - exposure time [yr]
  • param (physicsconstants) - set of physical parameters to be used.
Returns: float
dPhi/dE_nu at Icecube or Phi(E) if total flux is given.
